## WebSSH 線上終端機

WebSSH 是一個可以在瀏覽器裡操作的 SSH 終端機。你只需要開啟網頁，就能管理多組連線、切換主機、甚至同時保持多個工作階段。所有連線資料（包含私鑰）都只儲存在你的瀏覽器，不會上傳到伺服器。

### 可以做什麼？

- **集中管理連線**：把常用的主機、埠號、帳號、驗證方式都保存起來，下次只要點一下就能連線。
- **支援帳號密碼或憑證**：匯入 OpenSSH / PuTTY 私鑰時，只要提供 passphrase，系統會在瀏覽器端加密留下來，日後連線只需再輸入密碼。
- **多工作階段並存**：建立第二個、第三個連線也沒問題，畫面右側會列出所有進行中的工作階段，隨時切換或中斷。
- **無需安裝**：僅需一個現代瀏覽器（Chrome、Edge、Firefox…）即可使用。

### 基本操作

1. 開啟應用程式後，先在「連線清單」點「新增連線」，填寫：
   - 連線名稱、主機（Host）與埠號（Port）
   - 登入帳號（Username）
   - 驗證方式：帳號密碼或憑證
2. 若使用憑證，貼上私鑰或上傳檔案並輸入私鑰密碼（passphrase）。系統會先在瀏覽器端加密並保存。
3. 回到清單按「連線」，視窗會跳出驗證資訊：
   - 密碼登入：輸入帳號密碼即可
   - 憑證登入：若已儲存私鑰，只需輸入 passphrase；沒有儲存時可貼上或上傳新的私鑰
4. 連線成功後會切換到終端機畫面。需要同時連不同主機？再回到清單點一次就能新增另一個工作階段。

### 常見問題

**我的私鑰安全嗎？**  
私鑰只會存放在瀏覽器 LocalStorage，並以你輸入的 passphrase 透過 PBKDF2 + AES-GCM 加密。伺服器端無法取得原始內容。

**忘記 passphrase 怎麼辦？**  
因為系統不會保存你的密碼，只能重新貼上或上傳原始私鑰，再設定新的密碼保存。

**如何切換進行中的連線？**  
在連線清單下方會有「進行中的連線」卡片，點「切換」即可回到指定的工作階段，也能按「中斷」快速結束。

**清除連線資料？**  
在瀏覽器清除 LocalStorage 或點編輯後按「刪除私鑰」即可。若要完全歸零，可在瀏覽器的網站資料設定中刪除本站資料。

### 建議環境

- Chrome / Edge / Firefox / Safari 最新版本
- 啟用 JavaScript、LocalStorage
- 若使用憑證，請確保私鑰具備 passphrase
