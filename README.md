## WebSSH 線上終端機

這是一個在瀏覽器裡就能操作 SSH 的小工具。不需要安裝軟體，只要輸入連線資訊，就能像在終端機一樣遠端登入伺服器。所有連線設定與私鑰都儲存在你的瀏覽器，伺服器端不會保留任何資料。

### 功能亮點

- **連線管理**：把常用主機的 Host、Port、帳號和驗證方式全部記錄起來，下次一鍵連線。
- **支援帳號密碼與憑證**：匯入 OpenSSH / PuTTY 私鑰時，只要設定 passphrase，系統會在瀏覽器端加密保存，之後只需輸入密碼即可使用。
- **多工作階段並存**：同時連到多台主機也沒問題，畫面會列出所有進行中的連線，隨時切換或中斷。
- **純前端運作**：資料都在瀏覽器中處理，不用擔心私鑰上傳到他人伺服器。

### 安裝

```sh
npm install
```

### 開發模式

```sh
npm run dev
```

服務預設監聽 `http://localhost:8080`，會自動提供 `public/` 中的靜態檔案，可直接在瀏覽器開啟測試。

### 使用方式

1. **新增連線**
   - 在「連線清單」點 **新增連線**。
   - 填寫連線名稱、Host、Port、使用者帳號，選擇驗證方式。
   - 若使用憑證，貼上私鑰或上傳檔案並輸入 passphrase。沒有密碼的私鑰不會被儲存。
2. **儲存憑證（可選）**
   - 提供私鑰與 passphrase 後，系統會在瀏覽器端加密保存。若只想臨時使用，也可以不儲存。
3. **開始連線**
   - 回到清單點 **連線**。
   - 帳號密碼登入：輸入密碼即可。
   - 憑證登入：若已儲存私鑰只需輸入 passphrase；若未儲存，貼上或上傳私鑰再輸入 passphrase。
4. **切換或結束**
   - 多個連線同時進行時，可在「進行中的連線」卡片看到所有工作階段，點 **切換** 回到終端機，或點 **中斷** 結束連線。

### 常見問題

- **私鑰會上傳到伺服器嗎？**  
  不會。私鑰只會儲存在你的瀏覽器，並以你輸入的 passphrase 使用 PBKDF2 + AES-GCM 加密。

- **如何清除連線資料？**  
  編輯連線時可以刪除私鑰；若想完全歸零，可在瀏覽器的「網站資料」中刪除本系統存放的資訊。

- **需要什麼環境？**  
  Chrome、Edge、Firefox、Safari 等現代瀏覽器即可。請確保瀏覽器允許 JavaScript 與 LocalStorage。
